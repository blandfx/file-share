File Share Documentation

Main File Browser

GET /

Displays a list of files and directories in the configured media directory. Files are sorted by creation time in descending order.

File Download

GET/POST /download

Allows downloading media files from various sources using yt-dlp. Supports video and audio downloads from multiple platforms.

Mileage Tracker

GET/POST /mileage

Tracks vehicle mileage against lease terms, showing surplus miles and daily allowance calculations.

Notes Editor

GET/POST /notes

Provides a simple text editor for creating and editing .txt files in the media directory.

URL Control

GET/POST /url_control

Manages URLs for TV control, allowing assignment of channels to different URL slots.

Webcam Stream

GET /cam

Displays a webcam stream interface.

Most Recent File

GET /most_recent_file

Returns a JSON response containing the URL of the most recently added file in the media directory.

Device Blocking System

AUTO Device Blocking

Security feature that automatically blocks devices after 3 failed login attempts. Uses device fingerprinting (based on browser characteristics) to identify devices. Block duration starts at 1 hour and increases progressively with repeated blocks. Blocked device details are stored in blocked_devices.json. System administrators receive a Pushover notification when a device is blocked.

Blocked Devices Management

GET /blocked_devices

Displays a list of currently blocked devices, including their fingerprint ID, IP address, user agent, block duration, and block history. Requires authentication.

Device Unblocking

GET /unblock_device/<device_id>

Allows administrators to manually unblock devices using their unique device ID. Can be accessed directly or via the unblock buttons on the Blocked Devices Management page. Requires authentication.
